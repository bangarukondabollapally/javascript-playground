# Variables - Practice Challenges

## Challenge 1

Declare:

- Your name using `const`
- Your city using `const`
- Your age using `let`

---

## Challenge 2

Try this code and observe the result:

```js
let x = 5;
let x = 10;
```

**Question**

- What error do you get?
- Why?

---

## Challenge 3

Guess the output before running:

```js
console.log(count);

var count = 42;
```

Explain why.

---

## Challenge 4

Create a `const` object.

```js
const person = {
    name: "John"
};
```

Add a new property.

Does it work?

Why?

---

## Challenge 5

Access a `let` variable before declaring it.

```js
console.log(age);

let age = 20;
```

- What error do you see?
- Why does it happen?

---

## Challenge 6

Create a `const` array.

```js
const numbers = [1, 2, 3];
```

Push a new value.

Does it throw an error?

Explain why.

---

## Bonus Challenge

Predict the output before running.

```js
const a = 10;

{
    const a = 20;
    console.log(a);
}

console.log(a);
```
