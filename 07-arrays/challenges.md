## Challenge 7
Sort an array of ages in ascending order.

Predict the output first:
```js
[10, 2, 3].sort(); // ? — default sort() converts to strings, so order is unexpected
```
Then fix it using a compare function:
```js
arr.sort((a, b) => a - b); // correct numeric sort
```
