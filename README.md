# JavaScript Playground

> A beginner-friendly playground for learning JavaScript through runnable examples and practice challenges.

This repository focuses on learning through **runnable code**, **experiments**, and **practice challenges** instead of lengthy theoretical notes. Each chapter is self-contained, making it easy to revise concepts and understand JavaScript fundamentals by running the examples.

## Repository Structure

Each chapter follows the same structure:

```text
<topic>/
│
├── README.md
├── <topic>.js
├── challenges.md
└── index.html
```

* **README.md** – Quick reference for the chapter
* **`<topic>.js`** – Runnable examples, experiments, and challenge solutions
* **challenges.md** – Practice questions (no solutions)
* **index.html** – Runs the JavaScript in the browser

## Learning Philosophy

This repository follows a practical learning approach:

1. Learn the concept.
2. Experiment with runnable examples.
3. Solve the practice challenges.
4. Commit the chapter.
5. Move to the next topic.

The goal is to understand JavaScript by writing and running code rather than memorizing theory.

## Goals

- Learn JavaScript fundamentals through runnable code.
- Build a quick revision resource.
- Document my learning journey.
- Create a beginner-friendly JavaScript playground.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/bangarukondabollapally/javascript-playground
```

Open any chapter's `index.html` with **Live Server** (or any local server), then open your browser's Developer Console to view the output.
